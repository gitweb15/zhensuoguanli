1、状态查看操作 git status
2、添加操作 git add [文件名]
3、提交操作 git commit -m '提示信息' 【文件名】
4、查看历史记录:git log、git log --pretty=oneline、git log --oneline、git reflog
	            HEAD@{0}:移动当前版本需要多少步
5、版本前进、后退: git reset --hard [索引值]


《分支》
1、创建分支: git branch 【分支名】
2、查看分支: git branch -v
3、切换分支: git checkout 【分支名】
4、合并分支: 
	1、切换到接受修改的分支(被合并、新增新内容)上
		git checkout【被合并分支名】
	2、执行merge命令
		git merge【有新内容分支名】

《克隆1》
	git origin【远程地址】
1、完整的把远程库下载到本地
2、创建origin远程地址别名
3、初始化本地库

《通过git指令上传项目到github》
1、初始化  	git init
2、把文件添加到版本库中，使用命令 git add .添加到暂存区  git add .(文件下所有文件)
3、将文件提交到仓库	git commit -m "说明内容"
4、关联远程库	git remote add origin 【远程库地址】
5、将本地库的内容推送到远程库		git push -u origin master
